## 快速查场景

| 我要做什么 | 打开哪个文件 |
|-----------|------------|
| 加一个业务节点 | `prompts.py` + `recognizer.py` |
| 加一个 API 端点 | `server/api/` 下对应文件 |
| 调 LLM 输出格式 | `prompts.py` 对应 Node 的 prompt |
| 改数据真实性规则 | `config.py` 的 DATA_INTEGRITY |
| 加新 LLM 提供商 | `llm_client/` 新建 + `router.py` 注册 |
| 改前端某个页面 | `pages/` 下对应文件 |
| 加跨页面共享状态 | `context/AppContext.tsx` |
| 换 API 密钥 | `kaiwuback/.env` |
| 新建一个 Spec | 复制 `02-Spec模板.md` → 填完 → 更新 `03-项目文档索引.md` |
| 功能完成做 Review | 按 `04-AI-Review校验模板.md` 逐项检查 |
| 查某个决策为什么这么做 | `docs/specs/Spec-00X-xxx.md` |
| 不知道什么该做什么不该做 | `01-执行标准清单.md` |
