## 索引维护规则

1. **新增文档 → 同步新增索引行**：在任何分类下新增文档后，必须在本索引对应表格追加一行
2. **冻结状态变更 → 同步更新**：文档从"草稿"变为"已冻结"或反之，必须更新索引
3. **文档删除 → 索引行标记 `~~删除线~~`**：禁止直接删行，保留删除记录可追溯
